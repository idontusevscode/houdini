# AI-Assisted macros
This Houdini doc will help you understand AI-Assisted macros.

## Section 1 - Macro Files
Houdini stores macros in .hdi "Houdini Input Archive". These include the following files:
INPUTS.json
meta.hdinfo

## Section 2 - AI assisted macros?
AI-assisted macros are where the AI uses the macro, but then tailors it randomly like:
Original macro typed "AnonymousUnicorn", if you have Do random enabled when typing, it'll do a random name on some parts.
